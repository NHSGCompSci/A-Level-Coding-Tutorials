# A-Level Coding Tutorials

Welcome to the A-Level Coding Tutorials repository! This repo is designed to support and track the progress of students learning Python.

## Getting Started

1. **Clone or Fork** this repository to work on your own copy or directly within a Codespace.
2. **Assignments** are located in the `assignments/` directory. Follow the instructions in each assignment.
3. **Resources** are available in the `resources/` directory for additional support.
4. See `CONTRIBUTING.md` for collaboration and submission guidelines.

## Codespaces

Click the **Code** button and choose **Create codespace on main** to get started right away in a cloud-based environment with Python pre-installed.
